### How Mining Works

- **Free Play**: 1 game per 24 hours at standard mining rate
- **Boost Play**: Pay $EGG for extra games at 3x mining rate
- Each session gets a **luck roll** (0.4x–1.6x) + **skill factor** (accuracy × combos)
- Epoch pool halves every 7 days — early miners earn more
- More players = smaller individual share (difficulty adjustment)

### Deflationary Loop

```
Player pays $EGG for Boost → 50% burned (supply shrinks) + 50% recycled (extends mining)
                                    ↓
                         Fewer $EGG in existence
                                    ↓
                         Scarcity increases over time
```
